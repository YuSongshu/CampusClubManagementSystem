# 校园社团管理系统

## 概述

纯 JavaSE+JDBC+MySQL 开发，控制台运行，无任何框架



## 项目结构

```
src
├─ db.properties        
├─ utils
│  ├─ tool.java
│  └─ DbUtils.java       
├─ dao
│  ├─ UserDao.java
│  ├─ ClubDao.java
│  ├─ ApplyDao.java
│  ├─ ActivityDao.java
│  └─ ActivitySignDao.java
└─ Main.java   
```



## 本地部署

修改db.properties配置文件完善MySQL的driver，url，password
